这是 Mello's Blog 的第一篇文章。

我希望这个博客先服务于长期写作，而不是一开始就做复杂功能。以后可以把零散笔记逐步整理成几个方向：

- 考研学习：数学、英语、专业课、周复盘
- 信号与系统：概念、公式、例题和错题
- AI 与通信：论文、实验和工程想法
- 编程项目：环境搭建、问题记录、复盘总结

## 为什么用 Markdown

Markdown 足够轻量，适合快速记录，也方便以后迁移。比如一段代码可以这样写：

```python
def hello_blog(name: str) -> str:
    return f"Hello, {name}!"

print(hello_blog("Mello"))
```

## 先保持简单

这个博客暂时不做评论、登录、后台管理和复杂交互。先把写作路径打通：新建文章、预览、提交、自动部署。
